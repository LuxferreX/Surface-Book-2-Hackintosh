#🍎Surface Book 2 Hackintosh#

##设备配置💻##

   CPU:i5 7300u
   GPU：HD620
   RAM:16G(DIY)
   硬盘：512g（DIY）
   无线网卡：Comfast CF 812AC


##安装注意事项⚠️##

   1.禁用内置无线网卡和蓝牙
   2.将安全启动选项设置为Microsoft&3rd即可（绕过开机红锁）
